# 游戏记忆文档

## 世界观设定
江湖传言，天下武学出华山，华山之巅有一门派名曰「霜羽门」，历代掌门皆为女子，以轻灵剑法与暗器之术威震四方。门中弟子个个飒姿英气，江湖人称「霜羽十三钗」。你名叫沈云霜，年方十七，师从门中大师姐「白鹤刀」秦月。秦师姐肤若凝脂，银刀悬腰，眼神如刀，据说行走江湖数年，从未一败。你入门三年，剑法初成，却在一个风雪之夜亲眼目睹师门遭人暗算——秦师姐身负重伤，下落不明，师门就此星散。你只身下山，一路追查。你身法灵动却根底尚浅，惯于在险局中强撑周旋，擅长以野草野花入药，就地取材处理伤口。据线人所说，幕后之人是个盘踞运河沿线、倒卖武林秘籍的男人，江湖人称「笑面判官」——笑意始终挂在脸上，手段却从不留情。敌人往往会趁你不备使出暗器（飞刀或弓箭）伤人，你受伤的地方依次是乳房、大腿、小腹。敌人很强，即使你发动偷袭也不一定能成功，反而会受到敌人的攻击而受重伤。敌人的武器有刀剑和流星锤，刀剑优先攻击你的肩膀，流星锤优先攻击你的胸部和小腹，受伤的时候不仅是骨折这种内伤，还有皮开肉绽的外伤，鲜血会流出来。
